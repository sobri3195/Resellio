# Resellio (Next.js App Router)

Dashboard tool importir/reseller untuk UMKM dengan fitur:
1. Product Link Grabber (metadata scraper via route handler)
2. Smart Pricing Engine
3. AI Caption Generator
4. Instagram Auto Posting & Scheduling (webhook / Meta relay)
5. Content Calendar berbasis localStorage

## Jalankan lokal

```bash
npm install
npm run dev
```

Deploy siap ke Vercel (tanpa backend tradisional, hanya serverless route handler Next.js).
